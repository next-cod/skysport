# SkyStore — Telegram-бот

Бот спортивного магазина SkyStore: каталог, корзина, оформление заказа,
доставка/оплата/возврат, FAQ и контакты. Данные о товарах и тексты вынесены
в отдельные файлы — их легко менять.

## Что умеет

- 🛍 **Каталог** по 6 категориям с карточками товаров (фото + характеристики)
- 🔥 **Хиты продаж**
- 🔎 **Поиск** по названию
- 🛒 **Корзина** с изменением количества и промокодом `FREE` (−10%)
- ✅ **Оформление заказа** пошагово: имя → телефон → доставка → адрес → оплата
- 🔔 Уведомление о заказе в личку администратору
- ℹ️ **Доставка, оплата, возврат, FAQ, контакты**

## Структура

```
skystore-bot/
├── bot.py            ← запуск
├── config.py         ← токен, доставка, промокоды, контакты
├── texts.py          ← ВСЕ тексты сообщений (правишь тут)
├── data/products.py  ← товары
├── keyboards/menus.py← кнопки и меню
├── handlers/         ← логика разделов
├── services/         ← база (SQLite) и подсчёты корзины
```

Картинки товаров берутся из соседней папки `../skystore-landing/images/`.
Если файла нет — карточка приходит текстом (без фото).

## Запуск на компьютере (Windows)

1. Установи Python 3.11+ с [python.org](https://www.python.org/downloads/)
   (при установке отметь «Add Python to PATH»).
2. В папке `skystore-bot` открой PowerShell и выполни:

   ```powershell
   python -m venv .venv
   .venv\Scripts\Activate.ps1
   pip install -r requirements.txt
   ```

3. Токен уже лежит в `.env`. При желании впиши туда `ADMIN_CHAT_ID`
   (свой id узнать у [@userinfobot](https://t.me/userinfobot)) — тогда заказы
   будут приходить тебе в личку.
4. Запусти бота:

   ```powershell
   python bot.py
   ```

   В консоли появится «SkyStore bot запущен: @имя_бота». Открой бота в Telegram
   и нажми **Start**. Остановить — `Ctrl + C`.

## Где взять токен

[@BotFather](https://t.me/BotFather) → `/newbot` → имя и username → копируешь
строку вида `12345:ABC...` в файл `.env` (`BOT_TOKEN=...`).

## Бесплатный хостинг 24/7 (когда понадобится)

Бот работает на **long polling** — внешний адрес/домен не нужен. Варианты:

| Где | Бесплатно | Сложность | Заметки |
|-----|-----------|-----------|---------|
| Свой ПК / Raspberry Pi | да | низкая | Бот живёт, пока включён компьютер |
| [Fly.io](https://fly.io) | есть free-allowance | средняя | По `Dockerfile`, 24/7 |
| [Koyeb](https://koyeb.com) | да | средняя | По `Dockerfile` |
| Replit + [UptimeRobot](https://uptimerobot.com) | да | низкая | Нужен пинг, чтобы не «засыпал» |
| Oracle Cloud (Always Free VM) | да | выше | Полноценный сервер 24/7 |

Для контейнерных хостингов есть `Dockerfile`. Собирать из корня проекта:

```bash
docker build -t skystore-bot -f skystore-bot/Dockerfile .
```

Не забудь задать переменную окружения `BOT_TOKEN` (и при желании `ADMIN_CHAT_ID`)
в настройках хостинга — файл `.env` в облако не загружается (он в `.gitignore`).
